# 현인무 졸업작품
## 1. 코딩 환경 설정
### 가상환경
저희는 `pip install ~~`이런 명령어를 사용하는 라이브러리를 많이 사용하여 안전성과 호환성을 중시하여 가상환경을 설치하였습니다.
- 가상환경 설치
```
python3 -m venv venv
```
- 가상환경 활성화
```
source venv/bin/activate
```
- 가상환경 비활성화
```
deactivate
```
