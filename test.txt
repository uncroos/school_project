예시입니다.
깃허브에 연동되는지 실험중

-------------------------------

라즈베리파이에서 깃허브(GitHub) 연동하는 방법을 정리해줄게.  

---

### 🔗 **라즈베리파이에서 GitHub 연동하는 방법**  

### 1️⃣ **Git 설치**  
라즈베리파이에 Git이 설치되어 있는지 확인하고, 없으면 설치해야 함.  
```bash
git --version
```
설치되어 있지 않다면, 아래 명령어 실행  
```bash
sudo apt update
sudo apt install git -y
```

---

### 2️⃣ **Git 설정 (사용자 정보 등록)**  
Git을 사용할 때, 커밋할 유저 정보를 등록해야 함.  
```bash
git config --global user.name "너의 깃허브 아이디"
git config --global user.email "너의 깃허브 이메일"
```

---

### 3️⃣ **SSH 키 생성 및 GitHub에 등록**  
GitHub에 SSH 키를 등록하면 매번 로그인 없이 푸시(Push) 가능.  

#### **(1) SSH 키 생성**  
```bash
ssh-keygen -t ed25519 -C "너의 깃허브 이메일"
```
👉 엔터(Enter) 계속 눌러서 기본 경로(`~/.ssh/id_ed25519`)에 저장  

#### **(2) SSH 에이전트 실행 및 키 추가**  
```bash
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
```

#### **(3) SSH 공개키 확인 후 복사**  
```bash
cat ~/.ssh/id_ed25519.pub
```
👉 나오는 키를 복사  

#### **(4) GitHub에 SSH 키 등록**  
1. GitHub 접속 → [Settings](https://github.com/settings/profile)  
2. 왼쪽 메뉴에서 **SSH and GPG keys** 클릭  
3. **New SSH key** 버튼 클릭  
4. **Title** 아무거나 입력  
5. **Key**에 아까 복사한 키 붙여넣기  
6. **Add SSH key** 클릭  

#### **(5) SSH 연결 테스트**  
```bash
ssh -T git@github.com
```
👉 `Hi <username>! You've successfully authenticated...` 메시지가 뜨면 성공  

---

### 4️⃣ **GitHub 저장소 클론 (Pull)**  
원하는 GitHub 저장소를 라즈베리파이로 가져오려면 아래 명령어 사용  
```bash
git clone git@github.com:너의깃허브아이디/저장소이름.git
```

---

### 5️⃣ **변경사항 업로드 (Push)**  
#### **(1) 변경 사항 확인**  
```bash
git status
```

#### **(2) 변경된 파일 추가**  
```bash
git add .
```

#### **(3) 커밋 생성**  
```bash
git commit -m "변경 내용 작성"
```

#### **(4) GitHub에 업로드 (푸시)**  
```bash
git push origin main
```
(만약 브랜치 이름이 `main`이 아니면 `git branch`로 확인하고 맞게 변경)  

---

이제 라즈베리파이에서 깃허브랑 연동해서 코드 관리하면 됨. 🚀  
추가로 CI/CD 자동 배포도 설정하면 더 편하게 관리 가능함. 👌
